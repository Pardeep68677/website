# **App Name**: AapakaNai

## Core Features:

- Service Listings: Show all services like cutting (100rs), shaving (50rs), head massage (180rs), fire cutting (200rs), facial/grooming (250+GST) clearly.
- Image Gallery: Display images of services: Bollywood heroes, old heroes, Fauji cutting styles (no AI generated images).
- Customer Reviews: Allow customers to post reviews about their customer experience.
- Queue Prediction Tool: Estimate the waiting time on Sundays to manage customer flow, taking into account historical data.
- Contact Information: Display phone number and address

## Style Guidelines:

- Primary color: Deep saffron (#FF9933) to invoke an Indian cultural feel with warmth.
- Background color: Very light cream (#FAF9F6), a desaturated tint of the primary.
- Accent color: Earthy red-brown (#A66321), which is analogous to saffron, adding depth and a touch of retro style.
- Font: 'Literata', a serif font suitable for both headlines and body, lending a slightly vintage look
- Use small, hollow, transparent icons of scissors, razor blade, shaving cream, and other barber tools in a modern style.
- Design a clean, modern layout with an Indian cultural touch. High-quality photos and careful spacing.
- Subtle transitions for a smooth user experience.
